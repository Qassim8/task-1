This project is a simple UI built with React and Bootstrap, focusing on clean code, reusable components, and fully responsive design.

#Key Concepts Used

Components: Components like Navbar, Pricing, Contact, FormContact.

Reusable Component: PricingCard was built to be flexible and used multiple times without repeating code.

Props: Data is passed through props to keep components dynamic and maintainable.

Responsive Design: Bootstrap grid classes were used (row, col-sm, col-md, col-lg and gap) to ensure the layout works perfectly on mobile, tablet, and desktop.

Flexbox with Bootstrap: Utility classes such as (d-flex, align-items, justify-content and gap).

Mapping Data: Pricing plans and features were stored in arrays and rendered using .map() to make the structure easy to update.

#Technologies

React, Bootstrap, JS, CSS, react-icons
